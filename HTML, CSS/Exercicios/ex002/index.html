Paragrafos e Quebras de Linha

Voce pode escrever um paragrafo de qualquer jeito, basta colocar o texto enre <p> e </p> para iniciar e finalizar o paragrafo.

No caso de voce querer fazer uma quebra no paragrafo será necessario criar uma marcação para isso, como a que segue,
como pode ver mudou de linha o texto.
